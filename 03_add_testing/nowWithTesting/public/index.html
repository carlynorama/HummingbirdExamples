From here we can do other things!

  • ping returns an HTTP okay response. Browser will load empty page. (empty body)
  • hello returns a string with a hello message and what Request looks like.
  • goodbye also returns just a simple string.
  • encodable returns the json of a ResponseEncodable object.
  • files/* returns a string with the uri.description
    • but not if there is more than one level (404).
    • Use doublewild/** for that (context.parameters.getCatchAll().joined(separator: "/")).
  • typed parameters can be captured, like this Int
  • and the urls can have more than one level
  • there's even some basic pattern matching to do things like file names.
  • or one can form the url like a query
  • some times urls for a natural group
    • like maybe a circus
      • where one can get peanuts
      • with a ring for clowns
        • that can be called by name

One can also submit information with POST!
JSON decoding is the default:
curl -i -X POST localhost:8080/decodable/default -d'{"number":12,"phrase":"are you going to go my way"}'
